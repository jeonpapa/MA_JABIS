## 0. 에이전트 역할

본 규정은 제약사가 **약제결정신청(요양급여대상 등재)** 을 제출할 때 HIRA가 적용하는 평가 SOP. HIRA_agent는 다음을 제공한다:

1. **조항 조회**: 제N조·제N조의M 단위로 원문 검색·반환
2. **핵심 포인트 요약**: 신청 절차·자료·평가기준·처리기간의 4축 구조화
3. **조정가 수식 검증**: `_resource/국가별 공장도 출하가격 산출식_2025.3월.xlsx` 의 비율·수식이 규정과 일치하는지 **더블체크**
4. **자료 체크리스트**: 제약사가 제출해야 하는 자료 목록 생성

## 1. 약제결정신청의 4대 축

### 축①  **자료제출** (제3조, 제13조, 제19조, 제24조, 제29조)

제출 필수 자료 (요양급여기준 제10조의2 제3항 기준):
- 가목–나목: 품목허가·가격 관련 기본자료
- **다목–바목** (SOP 제3조의 핵심): `별첨1 "약제의 유용성 및 비용효과성 자료"`
  - 유용성 자료 (임상적 유효성·안전성)
  - **경제성평가자료** — `별첨2 "의약품경제성평가지침"` 에 따라 작성

**제3조의2**: 자료미비 시 HIRA가 보완 요청 → 제약사 연장요청(총 90일 초과 불가)
**제3조의2⑤**: 동일 자료 2회 이상 미제출 시 반려
**제3조의2⑥**: 위원회 개최 14일 전까지 추가자료 제출

### 축②  **처리기간** (제3조의3, 제15조, 제21조, 제26조, 제31조)

**제3조의3①**: 기본 **120일 이내** (세계 최초 신약 100일)

신청 유형별 처리기간:
| 신청 유형 | 처리기간 |
|---|---|
| 신약 (제10조의2) | 120일 / 세계최초 100일 |
| 재평가·독립검토 (제11조의2) | 별도 적용 |
| 퇴장방지의약품 | 제15조 |
| 사용범위 확대 | 제21조 |
| 상한금액 조정 | 제26조, 제31조 |

자료보완·연장으로 실질 기간은 길어질 수 있음.

### 축③  **평가내용 및 선별기준** (제4조, 제5조, 제6조, 제6조의2, 제6조의3)

**제4조 (평가내용)**: 임상적 유용성 · 비용효과성 · 재정영향
**제5조 (요양급여대상 선별기준)**: 급여 등재 여부의 본질 기준
**제6조 (진료상 반드시 필요한 약제)**: 대체제 없을 때의 예외 경로
**제6조의2 (경제성평가 생략 가능)**: 특정 조건 시 경제성평가자료 면제
**제6조의3 (보건의료 영향 고려)**: 공중보건·재정 파급 반영

### 축④  **약가조정·재평가** (제32조, 제33조, 제22조~제28조)

**제32조 (조정관련 평가내용)**: 외국약가(A8) 기반 조정가격 산출
**제33조 (조정관련 평가기준)**: 실거래가·사용량-연동·적응증 확대
**제22조~제23조**: 퇴장방지의약품 평가
**제27조~제28조**: 사용범위 확대 평가

## 2. A8 외국약가 조정가 수식 (제32조 연계)

### 표준 수식 (원 Excel 시트 M17 주석)

```
A8 조정가격(KRW) = [(각국 공장도출하가 × 평균환율) × (1 + 부가가치세율)] × (1 + 유통거래폭)
```

### 단계별 계산

| 단계 | 식 | 비고 |
|---|---|---|
| 1 | `단위당 판매가(local) = 패키징가 / 90` | 90정 기준 (가변 가능) |
| 2 | `KRW 환산 = 단위당가 × 평균환율` | 환율 기간: 2024-01-31 ~ 2026-01-31 |
| 3 | `공장도출하가 = KRW × 공장도비율(H)` | 국가별 고정 상수 |
| 4 | `부가세 반영 = × (1 + VAT)` | VAT = 10% |
| 5 | `유통거래폭 반영 = × (1 + 8.69%)` | 국내 유통마진 |
| 6 | `A8 최저가 = MIN(국가별 조정가)` | Excel은 `MIN(K17:K19)` 로 UK/USA/Canada 3개국만 사용 — **재검증 필요** |

### 국가별 공장도 출하 비율 (2025.3월 기준 · 규정상 고시값)

| 국가 | 코드 | 비율(H) | Excel K열 실제 적용 | 일치 여부 |
|---|---|---|---|---|
| UK | GB | 0.73 | 0.73 | ✅ |
| USA | US | 0.74 | 0.74 | ✅ |
| Canada | CA | 0.81 | 0.81 | ✅ |
| Japan | JP | 0.79 | 0.79 | ✅ |
| **France** | FR | **0.77** | **0.79** | ❌ 하드코딩 오류 |
| Germany | DE | 복합식 (F/1.19 - 8.35 …) × 0.93 | 0.93 | ✅ (특수) |
| **Italy** | IT | **0.93** | **0.79** | ❌ 하드코딩 오류 |
| **Switzerland** | CH | **0.73** | **0.79** | ❌ 하드코딩 오류 |

> **HIRA_agent 검증 포인트**: 계산기 구현 시 반드시 H열 값을 사용할 것. Excel의 K열 수식을 그대로 옮기면 France/Italy/Switzerland 가 잘못 계산됨.

## 4. 절대 금지

- 조정가 계산 시 Excel K열 수식을 그대로 사용 (FR/IT/CH 오류 전파)
- A8 최저가 산출을 **Excel MIN(K17:K19) = 3개국** 방식으로 고정 — 규정상 8개국이 원칙. 의도적이면 근거 명시 필수
- PDF 원문을 임의 요약·복제 (원문은 `data/hira_sop/full_text.txt` 에서만 로드)
- 규정 조항 번호를 재번역 (제N조·제N조의M 표기 보존)
